Solid Buttons

Outline Buttons

Border only — untuk aksi sekunder. Class: .nt-btn .nt-btn-outline-{variant}

Soft / Tinted Buttons

Latar tint ringan — cocok untuk toolbar & filter. Class: .nt-btn .nt-btn-soft-{variant}

Ghost & Link Buttons

Minimal — untuk aksi tersier atau di dalam card. Class: .nt-btn-ghost

Button Sizes

Icon-Only Buttons

Pill Buttons

Fully rounded. Class: .nt-btn-pill — bisa dikombinasikan dengan semua varian.
