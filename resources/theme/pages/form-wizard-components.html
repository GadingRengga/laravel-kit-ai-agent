Form Wizard — Vertical Stepper

Step list ditampilkan di sisi kiri dalam grid 2 kolom. Cocok untuk wizard checkout / order. Blade: <x-wizard style="vertical">

Siap diproses!

Form Wizard — Minimal (Progress Bar)

Progress bar tipis + judul step saat ini. Cocok untuk onboarding singkat atau survey. Blade: <x-wizard style="minimal">

data-style="minimal"
Step ini opsional — boleh dilewati tanpa diisi.

Referensi Attribute

Semua pengaturan wizard dilakukan lewat data-* pada elemen .nt-wizard — tidak perlu menulis JS tambahan.

Attribute Nilai Default Keterangan
data-style horizontal · vertical · minimal horizontal Tampilan navigasi step.
data-validate true · false true Validasi field required pada step sebelum lanjut.
data-allow-step-click true · false false Klik langsung pada header step untuk pindah (hanya step yang sudah dilewati).
data-allow-skip true · false false Lewati validasi untuk semua step, atau step dengan data-step-optional="true".
data-persist true · false false Simpan posisi step ke sessionStorage agar tidak hilang saat refresh.
data-persist-key string nt-wizard-{index} Key custom untuk sessionStorage, berguna jika ada beberapa wizard di satu halaman.
data-scroll-top true · false true Auto-scroll ke atas wizard saat pindah step.
data-show-step-count true · false true Tampilkan teks "Langkah X dari Y" (khusus style minimal).
data-labels-prev / -next / -submit string Sebelumnya / Selanjutnya / Simpan Label tombol footer.

Per-step (.nt-wizard-step)

Attribute Contoh Keterangan
data-step-title "Akun" Judul step (wajib).
data-step-subtitle "Email & password" Deskripsi singkat (opsional, tidak tampil di mobile pada style horizontal).
data-step-icon "fa-solid fa-user" Class FontAwesome, ditampilkan pada marker step yang belum aktif (digantikan nomor jika kosong).
data-step-optional true Step ini tidak divalidasi & diberi badge "Opsional".

Event JavaScript (untuk integrasi)

document.getElementById('wizard-register')
  .addEventListener('nt-wizard:submit', (e) => {
    console.log(e.detail.formData); // semua field di seluruh step
    // contoh: kirim via fetch ke endpoint Laravel
    // fetch('/api/register', { method: 'POST', body: JSON.stringify(e.detail.formData) })
  });

document.getElementById('wizard-register')
  .addEventListener('nt-wizard:step-change', (e) => {
    console.log(e.detail.from, '->', e.detail.to);
  });
